# Physics Foundations

## The Central Question

> Can mathematical abstractions (potentials) directly influence physical reality, or are they merely computational conveniences?

This project explores this question through visual art, making the invisible visible.

## Key Physical Quantities

### Forces vs Fields vs Potentials

| Concept | Type | Example | Visual |
|---------|------|---------|--------|
| **Force** | Vector | F = GMm/r² | Arrows pointing at objects |
| **Field** | Vector field | g = -GM/r² r̂ | Arrow field filling space |
| **Potential** | Scalar field | V = -GM/r | Height map / color gradient |

### The Hierarchy

```
Potential (scalar)
    │
    │ gradient (∇)
    ▼
Field (vector)
    │
    │ × mass/charge
    ▼
Force (vector on object)
```

## Why Potentials Matter for Art

1. **Scalars are easy to visualize** - Height maps, color gradients, isosurfaces
2. **Addition is trivial** - Superposition of multiple bodies just adds values
3. **Reveals hidden structure** - Lagrange points, equipotential surfaces

## The Three Potentials

### Gravitational Potential (V)
```
V = -GM/r
```
- Creates "wells" around massive objects
- Always negative (attractive only)
- Used by: Newton, Lagrange

### Electric Potential (φ)
```
φ = kQ/r
```
- Creates wells AND hills (attraction + repulsion)
- Can be positive or negative
- Developed by: Poisson (1810s)

### Magnetic Vector Potential (A)
```
B = ∇ × A  (curl of A)
```
- A is a VECTOR, not scalar
- Required because magnetic field lines form loops
- Developed by: Thomson/Kelvin (1840s)
- Central to Aharonov-Bohm effect

## Artistic Implications

| Potential | Visualization Approach |
|-----------|----------------------|
| Gravitational | 3D terrain, depth wells |
| Electric | Bipolar landscapes (valleys + peaks) |
| Magnetic | Swirling vector flows, vortices |

## Key Insight for Visualization

The same physical system can be represented as:
- A messy tangle of force vectors (hard to see patterns)
- A smooth potential landscape (easy to see structure)

**Our art will show both** - letting viewers toggle between representations to see how the "mathematical trick" reveals hidden beauty.
